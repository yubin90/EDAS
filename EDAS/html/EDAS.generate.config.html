EDAS.generate.config {EDAS}R Documentation

EDAS 시스템 등록용 모델 설정 파일생성

Description

EDAS 시스템에 모델 등록시 사용되는 모델 config 파일을 생성해 줍니다.

Usage

EDAS.generate.config(configPath = "~/model.conf")

Arguments

configPath

모델 설정(configuration) 파일을 저장 할 사용자 local directory (default=~/model.conf)
