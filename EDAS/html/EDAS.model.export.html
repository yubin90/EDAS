EDAS.model.export {EDAS}R Documentation

Export Model

Description

모델 파일을 공유폴더로 복사합니다.

Usage

EDAS.model.export(modelFilePath = NULL, name = NULL)

Arguments

modelFilePath

등록하고자 하는 모델 파일 경로

name

공유폴더에 저장되는 모델 파일명

See Also

Other Model: EDAS.model.ls

Examples

modelFilePath <- getwd()
modelName <- "train.dat"
EDAS.model.export(modelFilePath, modelName)
