edas_sample3 - 전력수요예측을 위한 샘플 데이터셋

Description

This is a basic sample dataset (triaining/testing dataset). The data consists of the following features: 일 - Day 시 - Hour 분 - Minute 사용량.kWh - PW (PowerUsage-kWh) 최대수요.kW - MPW (MaxPowerUsage-kW) 지상무효전력.kVarh. - GRP 진상무효전력.kVarh. - RP 지상역률... - GPF 진상역률... - RPF 요일 - DoW 근무여부 - isWorking 전일영향 - YI 기온 (℃) - T (Temperature) 습도 ( 엔탈피(kJ/kg) - E (enthalpy) HD1 CD1 HD2 CD2

Usage

data(edas_sample3)

Format

A list contains training and testing data sets with 3 variables
