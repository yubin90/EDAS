EDAS.getdata {EDAS}R Documentation

EDAS ETL (Extraction-Transformation-Loader) 추출한 학습(train)/테스트(test) 데이터 셋 로더

Description

EDAS ETL에서 추출된 데이터를 읽어 올때 사용합니다.

RStudio에서는 임의로 지정된 경로에 저장되어있는 EDAS ETL 데이터 파일들을 읽어오며, EDAS Analytics 엔진에서는 데이터를 argument로 받아 처리하게 되어있습니다.

'inputCols'를 사용하여 모델의 input 데이터 포맷을 임의로 정의 할 수 있으며 EDAS ETL에서 추출한 데이터를 읽어올때 사용됩니다.

*주의사항 - 정의된 컬럼 리스트와 ETL에서 추출한 데이터의 컬럼 길이가 다를 경우 오류를 발생시킵니다.
모델이 generic할 경우, 정의된 컬럼(inputCols)외에 추가적인 컬럼들에 대한 제악사항이 없을 경우 isModelGeneric 옵션을 통해 선정 해 줄수있습니다.

하나의 data.table로 결과를 받고 싶을 경우 "AsCombined" 값을 TRUE(default=FALSE)로 설정하여 호출하시면 됩니다.

Usage

EDAS.getdata(inputCols = NULL, isModelGeneric = TRUE, AsCombined = FALSE)

Arguments

inputCols

A character vector of strings - 모델이 지원하는 input 데이터(training/test) 포맷을 정의 합니다.

isModelGeneric

A boolean - 모델의 generic 여부 (기본값=TRUE)

AsCombined

A boolean - 하나의 데이터 테이블만 리턴 할 경우

Value

train - 학습 데이터 셋 test - 테스트 데이터 셋
