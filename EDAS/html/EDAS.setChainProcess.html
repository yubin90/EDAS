EDAS.setChainProcess {EDAS}R Documentation

EDAS 모델 연계성 설정 API

Description

EDAS 워크플로우(Workflow)상에서 다음 모델에게 전달할 학습/테스트 데이터를 정의 합니다. *참고 – 체인 프로세스(학습/테스트 데이터 셋)이 정의 되어 있지않을 경우, 워크플로우 상에서 등록되는 다음 모델은 데이터를 받을 수가 없게 됩니다 (오류 발생)

Usage

EDAS.setChainProcess(training = NULL, test = NULL)

Arguments

training

data.frame/data.table 포맷의 학습 데이터 셋

test

data.frame/data.table 포맷의 테스트 데이터 셋

See Also

Other EDAS: EDAS.clearAll, EDAS.commit, EDAS.generate.config, EDAS.getdata, EDAS.registerGraphicFunction, EDAS.setIntegration, EDAS.showggplot, EDAS.showtable

Examples

dataasets <- EDAS.getdata()
training_dataset <- datasets$train #data frame
testset_dataset <- datasets$test #data frame
# do something with the datasets

EDAS.setChainProcess(training=training_dataset, test=testset_dataset)
