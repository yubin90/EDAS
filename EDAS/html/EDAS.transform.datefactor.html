EDAS.transform.datefactor {EDAS}R Documentation

EDAS Data Transformation Method - date factor

Description

데이터를 주중/주말/특별한(Working/Non-Working/Special Days) 날로 구분 주중을 정의 하는 요일들(1~7) numeric list, 기본값으로 월요일부터 금요일 (1~5)

Usage

EDAS.transform.datefactor(dt, datetimeCol = NULL, workingdays = c(1, 2, 3,
  4, 5), specialdays = NULL)

Arguments

dt

data frame or data table

datetimeCol

datetime(POSIXct) column name

workingdays

list of numeric values

specialdays

list of character 특별한 날을 정의 하는 character list, 날짜 포맷 yyyy-mm-dd

Value

indextable

정의된 주중/주말/특별한날의 index 값

workingdays

정의된 업무요일 값(들)

nonworkingdays

정의된 비업무요일 값(들)

specialdays

정의된 특별한 날짜(들)

eval

input 데이터 셋(데이터 프레임)에 요일(dayOfweek) 값과 주중/주말/특별한 날을(datetype) 나타내는 값을 추가한 결과

See Also

Other EDAS-Data-Transformation: EDAS.transform.timefactor

Examples

#generate a sample dataset
dataset <- data.frame("Datetime"=seq(ISOdate(1900,1,1), ISOdate(1901,1,1), length.out=20),
"x"=seq(1,10,length.out=20))

EDAS.transform.datefactor(dataset, "Datetime")

specialdays <- c("2016-01-02", "2016-01-03")
EDAS.transform.datefactor(dataset, "Datetime", specialdays = specialdays )
