edas_sample2 - 전력수요예측을 위한 샘플 데이터셋

Description

This is a basic sample dataset (triaining/testing dataset). The data consists of the following features: 1. collect_date - 날짜시간 2. PowerUsage - 전력사용량 3. Temperature - 외부 온도 4. RH - 습도 5. Entropy

Usage

data(edas_sample2)

Format

A list contains training and testing data sets with 3 variables

See Also

Other EDAS-Data-Samples: edas_sample1, edas_sample3

Examples

data(edas_sample2)
trainData <- edas_sample2$train
testData <- edas_sample2$test
