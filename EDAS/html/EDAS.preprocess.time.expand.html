EDAS.preprocess.time.expand {EDAS}R Documentation

Data Preprocessing - Time Expansion

Description

datetime(POSIXct) 컬럼에서 날짜 범위(day)를 추출하여

주어진 데이터 수집주기 만큼 데이터를 생성합니다.

Usage

EDAS.preprocess.time.expand(dt, datetimeCol = NULL, mins = 15)

Arguments

dt

data frame or data table

datetimeCol

character datetime(POSIXct) column name

mins

quick_periods minutes ("lubridate")

Value

data.table with extanded time series

See Also

Other EDAS-Preprocess-missingdata: EDAS.preprocess.fillmissingdata

Examples

startT <- as.POSIXct("2015-01-01 00:00:00", format="%Y-%m-%d %H:%M:%S") #Start Time
endT <- as.POSIXct("2015-01-02 00:00:00", format="%Y-%m-%d %H:%M:%S") #End Time
#2015-01-01 ~ 2015-01-02 (interval=15mins, 97)
sampleDF <- data.frame(dt=seq(startT, endT, by="15 mins"), power=rnorm(97,0,0.1))
EDAS.preprocess.time.expand(sampleDF, datetimeCol="dt")

sampleDF2 <- data.frame(collect_date=seq(startT, endT, by="15 mins"), power=rnorm(97,0,0.1))
EDAS.preprocess.time.expand(sampleDF2)
