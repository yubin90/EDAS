EDAS.showtable {EDAS}R Documentation

모델 리포팅 - 테이블 데이터

Description

분석 결과, 리포팅에 사용될 data frame or data table을 등록합니다. 등록된 테이블 데이터는 commit(EDAS.commit)시 EDAS UI에 보여집니다.

Usage

EDAS.showtable(resulttable)

Arguments

resulttable

- data frame or data table

See Also

Other EDAS: EDAS.clearAll, EDAS.commit, EDAS.generate.config, EDAS.getdata, EDAS.registerGraphicFunction, EDAS.setChainProcess, EDAS.setIntegration, EDAS.showggplot

Examples

edasDF <- data.frame(x=c(1,2,3),y=c(1,2,3))
EDAS.showtable(edasDF)
