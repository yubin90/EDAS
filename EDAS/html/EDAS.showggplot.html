EDAS.showggplot {EDAS}R Documentation

모델 리포팅 - ggplot 연동

Description

분석 결과, 리포팅에 사용할 ggplot 기반의 챠트를 EDAS 시스템에 등록합니다. 등록된 챠트는 commit(EDAS.commit)시 EDAS 화면에 보여집니다.

Usage

EDAS.showggplot(resultplot)

Arguments

resultplot

- ggplot object

See Also

Other EDAS: EDAS.clearAll, EDAS.commit, EDAS.generate.config, EDAS.getdata, EDAS.registerGraphicFunction, EDAS.setChainProcess, EDAS.setIntegration, EDAS.showtable

Examples

library(ggplot2)
edasDF <- data.frame(x=c(1,2,3),y=c(1,2,3))
ggplotExample <- ggplot(edasDF) + geom_line(aes(x, y))
EDAS.showplot(ggplotExample)
