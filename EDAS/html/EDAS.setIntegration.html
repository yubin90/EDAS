SetIntegration - 모델 결과 연동 API

Description

모델 결과를 다른 시스템(예: BEMS)으로 연동 할 때 사용되는 API 입니다. 모델러는 모델 결과 연동시 사용되는 uniqueID 및 message(최종결과)를 정의해야 합니다. Message는 result.txt파일에 저장되며 Timestamp(실행시점) 와 Message 컬럼을 가지는 테이블 형태를 가지고 있습니다.

Usage

EDAS.setIntegration(uniqueID = NULL, message = NULL)

Arguments

uniqueID

연동결과를 식별하는 unique ID (예:제주한라병원 공조기 가동시간)

message

모델 결과(png,dat(=csv),log)외에 추가적으로 보내고자 하는 메세지 또는 값

See Also

Other EDAS: EDAS.clearAll, EDAS.commit, EDAS.generate.config, EDAS.getdata, EDAS.registerGraphicFunction, EDAS.setChainProcess, EDAS.showggplot, EDAS.showtable

Examples

EDAS.setIntegration("H병원_단기전력수요예측_피크타임", "2017-01-01 12:00:00")
