EDAS.registerGraphicFunction {EDAS}R Documentation

모델 리포팅 - 커스텀 그래픽 연동

Description

분석 결과, 리포팅에 사용할 커스텀 그래픽 함수를 EDAS 시스템에 등록합니다. 등록된 함수에서 생성되는 그래픽은 commit(EDAS.commit)시 EDAS 화면에 보여집니다.

Usage

EDAS.registerGraphicFunction(graphicFunction = NULL)

Arguments

graphicFunction

- 커스텀 그래픽(시각화) 함수

See Also

Other EDAS: EDAS.clearAll, EDAS.commit, EDAS.generate.config, EDAS.getdata, EDAS.setChainProcess, EDAS.setIntegration, EDAS.showggplot, EDAS.showtable

Examples

customGraphicFunExample <- function(){ plot(1,2) }
EDAS.registerGraphicFunction(customGraphicFunExample)
