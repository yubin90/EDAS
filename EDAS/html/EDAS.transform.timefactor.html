EDAS.transform.timefactor {EDAS}R Documentation

EDAS Data Transformation Method - time factor (Working/Non-Working Hours)

Description

데이터를 주중/주말/특별한(Working/Non-Working/Special Days) 날로 구분하는 값을 생성합니다. 주중을 정의 하는 요일들(1~7) numeric list, 기본값으로 월요일부터 금요일 (1~5)

Usage

EDAS.transform.timefactor(dt, datetimeCol = NULL, workinghours = 9:18)

Arguments

dt

data frame or data table

datetimeCol

datetime(POSIXct) column name

workinghours

list of numeric values - 업무시간을 정의 하는 list (0~23), 기본값으로 오전 9시부터 오후 6시(포함)까지

Value

indextable

정의된 주중/주말/특별한날의 index 값

workinghours

정의된 업무시간 값(들)

nonworkinghours

정의된 비업무시간 값(들)

eval

input 데이터 셋(데이터 프레임)에 요일(dayOfweek) 값과 주중/주말/특별한 날을(datetype) 나타내는 값을 추가한 결과

See Also

Other EDAS-Data-Transformation: EDAS.transform.datefactor

Examples

#generate a sample dataset
dataset <- data.frame("Datetime"=seq(ISOdate(1900,1,1), ISOdate(1901,1,1), length.out=20),
"x"=seq(1,10,length.out=20))

EDAS.transform.timefactor(dataset, "Datetime")
