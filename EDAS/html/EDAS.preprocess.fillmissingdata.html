EDAS.preprocess.fillmissingdata {EDAS}R Documentation

Data Preprocessing - Fill Missing Data

Description

결측치(missing data)를 처리하는 기본적인 함수들(전값/평균/다음값)을 제공하는 API 입니다.

Usage

EDAS.preprocess.fillmissingdata(dt, how = c("average", "prior", "next"))

Arguments

dt

- data frame or data table

how

- a method used to fill the missing data within the dataframe: prior(전값),average(평균) and next(다음값)

See Also

Other EDAS-Preprocess-missingdata: EDAS.preprocess.time.expand

Examples

## d
d <- EDAS.preprocess.fillmissingdata()
