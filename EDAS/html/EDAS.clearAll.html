EDAS.clearAll {EDAS}R Documentation

ClearAll

Description

R Studio 환경(session)에 저장된 모든 EDAS 결과물(챠트, 테이블, 학습/테스트 데이터셋 등)을 비웁니다. EDAS 결과(챠트,테이블,학습/테스트 데이터)들은 global option에 저장되며 이는 Environment 변수들과 같이 자동으로 session에 저장이 됩니다.

특정 모델(R Script)의 결과물(EDAS UI 상에서)들만 확인하고 싶으실 경우 이 함수를 사용해서 변수들을 비우고 실행시키시면 됩니다. *참고 이 함수는 RStudio 상에서만 동작 합니다, EDAS Workflow상에서는 동작하지 않습니다.

Usage

EDAS.clearAll()
