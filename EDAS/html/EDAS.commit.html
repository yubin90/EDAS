EDAS.commit {EDAS}R Documentation

EDAS 모델 commit

Description

모델에서 등록된 챠트/테이블/커스텀 그래픽(함수)/Chain 분석 모델 결과를 EDAS에 확인/검증 합니다. 결과파일들은

Usage

EDAS.commit(localDir = "~/edas-model-test")

Arguments

localDir

- a character, RStudio에서 실행시 모델 결과 파일들을 저장할 경로

See Also

Other EDAS: EDAS.clearAll, EDAS.generate.config, EDAS.getdata, EDAS.registerGraphicFunction, EDAS.setChainProcess, EDAS.setIntegration, EDAS.showggplot, EDAS.showtable

Examples

library(ggplot2)

edasDF <- data.frame(x=c(1,2,3),y=c(1,2,3))
ggplotExample <- ggplot(edasDF) + geom_line(aes(x, y))
EDAS.showplot(ggplotExample)
EDAS.showtable(edasDF)

EDAS.commit()
